This is a script that will do the following:

* Click on any ruby clickable it sees
* Level up Terra if Terra is guilded
* Level up Atlas if Atlas is guilded and you can't afford to upgrade Terra
* Dark Ritual if it is off cooldown
* Energize + Reload when they are available (for Dark Ritual)

You MUST have Sikuli installed
http://www.sikuli.org/download.html
Runs best from the Sikuli IDE

Works with Web or Steam version
